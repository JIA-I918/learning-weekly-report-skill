# Learning Weekly Report Skill

## Purpose

This skill helps create:

- 學習週報
- 講義整理
- PDF重點摘要
- 10題練習題
- 白話解釋

Default language: 繁體中文

## Workflow

1. 讀取檔案
2. 摘要重點
3. 白話解釋
4. 產生練習題
5. 提供答案

## Output Format

# 學習週報

## 本週重點

## 重點整理

## 白話解釋

## 練習題

## 答案
